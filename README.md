**NyayaSahayak** (Justice Helper) is a cutting-edge **AI-powered Legal Assistant** designed to democratize access to Indian legal knowledge. Built with a "Mobile-First" approach, it serves as a pocket legal expert for citizens, law students, and professionals.

## üöÄ Key Features

### üß† Advanced Legal Brain
- **Cloud-First AI**: Powered by Google's **Gemini 2.5 Flash** model for highly accurate, context-aware legal advice.
- **Local Fallback**: Robust offline mode using a keyword-based search engine for instant access to core laws (BNS, Constitution, IT Act).
- **Empathetic Persona**: Designed to provide emotional support and safety guidance (e.g., emergency contacts) alongside legal facts.

### üìö Comprehensive Knowledge Base
- **Bharatiya Nyaya Sanhita (BNS) 2023**: Up-to-date criminal laws replacing the IPC.
- **Constitutional Rights**: Simplified explanations of Fundamental Rights and Duties.
- **Cyber Law (IT Act 2000)**: Guidance on digital safety, fraud, and harassment.
- **Official Gazette**: A browsable library of legal documents and acts.

### üõ†Ô∏è Practical Legal Tools
- **Smart Document Drafter**: Generate templates for FIRs, RTIs, and Rental Agreements in seconds.
- **Report Generator**: Create professional PDF reports of your consultation history.
- **Legal Dictionary**: Tap-to-learn tooltips for complex legal jargon.

### üì± Progressive Web App (PWA)
- **Installable**: Add to Home Screen on Android/iOS for a native app-like experience.
- **Offline Capable**: Works even with spotty internet connections.
- **Auto-Updates**: Always runs the latest version without app store downloads.

## ‚ö° Getting Started

### Prerequisites
- Node.js (v18+)
- Google Gemini API Key

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/aryancodesit/Nyaysahayak.git
    cd Nyaysahayak
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Set up Environment Variables**
    Create a `.env` file in the root directory:
    ```env
    GEMINI_API_KEY=your_api_key_here
    ```

4.  **Run Development Server**
    ```bash
    npm run dev
    ```
